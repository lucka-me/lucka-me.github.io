PROJECT LUNATIC

================================================================================
DESCRIPTIONS
================================================================================
Project Lunatic is a secret project under Lunatic Lab's command and
responsibility. The lab and project is fully supported by Lucka Lab.
The goal of this project is to build and maintain a virtual sub-prototype from a
real humanity prototype.
The virtual sub-prototype called Lunatic Prototype is used to replace the real
one in some dangerous fields to protect the safety of the real one.
